。……　　好多年后，我似乎全然忘却了我那可爱的风筝。有一年年关时候，家里收拾房屋，意外地拿出了那个曾经被我遗弃的风筝，破烂的身躯又落了厚厚的灰尘。这么多年，父亲一直让他留在了老屋深处。烧掉吧，我说，那时候我还能希求什么呢？　　后来，我模糊的记得，那个放风筝的青年人在一个深夜的酒后去了他的风筝远去的地方。　　那时，村头高压线上缠绕的风筝线已经飘摇了很多年，但可能已经好像不在了。　　　　　　　　　　　　　　鲁青　　　　　　　　　　　2006年3月19日于西凉
